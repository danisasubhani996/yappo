# 💬 Realtime Chat App

A sleek realtime private messaging application built with Socket.IO and Node.js — focused on smooth interactions, modern UI, and responsive performance.

---

## ✨ Highlights

- Realtime private messaging
- Online / offline presence
- Typing indicators
- Message reactions
- Reply & forward messages
- Delete for me / everyone
- Read receipts
- File sharing
- Audio messages
- Image preview viewer
- Dark / Light mode
- Mobile responsive design

---

## 🛠 Tech Stack

### Frontend
- HTML5
- CSS3
- Vanilla JavaScript

### Backend
- Node.js
- Express.js
- Socket.IO

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/danisasubhani996/yappo.git
```

Move into the project folder:

```bash
cd yappo
```

Install dependencies:

```bash
npm install
```

Start the server:

```bash
node server.js
```
